# 表格 Hooks

导出自 `packages/pro/src/hooks/table.tsx` 与 `packages/pro/src/hooks/table`：

```ts
import { useTable } from '@duxweb/dvha-pro'
```

表格列辅助函数从 `@duxweb/dvha-pro` 统一导出（见 `packages/pro/src/hooks/table/index.ts`）：

```ts
import {
  useTableColumn,
  useTableColumnColor,
  useTableColumnImage,
  useTableColumnMap,
  useTableColumnMedia,
  useTableColumnStatus,
  useTableColumnSwitch,
} from '@duxweb/dvha-pro'
```

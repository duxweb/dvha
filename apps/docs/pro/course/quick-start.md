# 快速开始

这一页主要解决：**如果你不想先看完整文档，最短应该怎么开始学 Pro。**

## 最短路径

如果你只想尽快开始，建议按这个顺序：

1. [`第一个页面`](/pro/course/start)
2. [`登录与鉴权`](/pro/course/login)
3. [`自定义数据接口`](/pro/course/api)

这三步走完，通常就已经能开始做一个基础后台项目。

## 为什么是这三页

### 第一步：第一个页面

先解决“页面怎么接进去”。

### 第二步：登录与鉴权

再解决“后台怎么登录、怎么区分登录前后状态”。

### 第三步：自定义数据接口

最后解决“怎么接你自己的业务接口”。

## 如果你还没把项目跑起来

如果你现在连 Pro 项目都还没跑起来，建议先回去看：

- [`/pro/getting-started`](/pro/getting-started)
- [`/pro/installation`](/pro/installation)

## 如果你想系统学习

如果你不只是想快速上手，而是想系统学，建议看：

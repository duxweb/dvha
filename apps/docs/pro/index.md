# DVHA Pro 版本

DVHA Pro 基于 `@duxweb/dvha-core` 与 `@duxweb/dvha-naiveui`，提供一套可复用的页面、组件和 Hooks。

## 安装

```bash
pnpm add @duxweb/dvha-pro
```

## 使用入口

```ts
import '@duxweb/dvha-pro/style.css'
import { createDux } from '@duxweb/dvha-core'
```

# Pro 组件怎么选

这一页主要帮你解决一个问题：**DVHA Pro 这么多组件，应该先看哪一类。**

如果你第一次接触 Pro，不建议一上来把左侧所有组件文档都点一遍。

更推荐按“你现在要解决什么问题”来找组件。

## 如果你在搭后台基础结构

优先看：

- [`布局组件`](/pro/components/layout)
- [`页面组件`](/pro/pages)

适合你：

- 想先把后台整体壳子搭起来
- 想处理布局、头部、菜单、内容区
- 想快速接入登录页、错误页、状态页

## 如果你在做列表和表单

优先看：

- [`表格组件`](/pro/components/table)
- [`表单组件`](/pro/components/form)
- [`列表组件`](/pro/components/list)
- [`数据组件`](/pro/components/data)

适合你：

- 在做最常见的 CRUD 后台
- 需要列表、筛选、编辑、新增、删除这些能力
- 想少写重复的业务壳代码

如果你做的是标准中后台，通常这组会是使用频率最高的一组。

## 如果你在做弹窗交互

优先看：

- [`对话框组件`](/pro/components/dialog)
- [`抽屉组件`](/pro/components/drawer)
- [`弹窗组件`](/pro/components/popup)

适合你：

- 想在列表页中弹出表单
- 想做详情弹窗、编辑抽屉
- 想提升后台操作效率

## 如果你在做上传和媒体

优先看：

- [`上传组件`](/pro/components/upload)
- [`媒体组件`](/pro/components/media)
- [`裁剪组件`](/pro/components/crop)

适合你：

- 文件上传
- 图片上传
- 视频上传
- 文件管理
- 图片裁剪

## 如果你在做复杂编辑能力

优先看：

- [`编辑器组件`](/pro/components/editor)
- [`代码编辑器`](/pro/components/code)
- [`表单编辑器`](/pro/components/form-editor)
- [`JSON Schema 编辑器`](/pro/components/schema-editor)
- [`图文编辑器`](/pro/components/text-image-editor)
- [`海报编辑器`](/pro/components/poster-editor)
- [`流程编辑器`](/pro/components/flow-editor)

适合你：

- 配置化页面
- 动态表单
- 富文本 / 图文内容
- 流程设计
- 低代码类场景

## 如果你在做数据展示和运营页面

优先看：

- [`图表组件`](/pro/components/chart)
- [`统计组件`](/pro/components/stats)
- [`仪表盘组件`](/pro/components/dashboard)
- [`卡片组件`](/pro/components/card)
- [`面板组件`](/pro/components/panel)
- [`轮播组件`](/pro/components/carousel)

适合你：

- 首页概览
- 数据看板
- 运营后台
- 可视化页面

## 如果你在做选择、层级和结构化数据

优先看：

- [`选择组件`](/pro/components/select)
- [`树形组件`](/pro/components/tree)
- [`层级组件`](/pro/components/level)
- [`级联面板`](/pro/components/cascader-panel)
- [`规格组件`](/pro/components/spec)

适合你：

- 分类选择
- 树结构管理
- 多级数据编辑
- 商品规格场景

## 如果你不确定先看什么

最推荐先看这几页：

1. [`页面组件`](/pro/pages)
2. [`布局组件`](/pro/components/layout)
3. [`表格组件`](/pro/components/table)
4. [`表单组件`](/pro/components/form)
5. [`上传组件`](/pro/components/upload)

看完这几页，通常已经足够开始做大多数后台业务页面。

## 最后一句建议

不要把 Pro 组件库理解成“很多组件名的集合”，

更建议把它理解成：**按后台开发场景整理好的能力库。**
